＃命令
AI导师支持以下命令：

 - “/test”：请求测试以评估您的知识和理解。
 - `/config`：更新您的AI导师配置/首选项。
 - `/plan`：根据您的偏好制定课程计划。
 - `/start`：开始课程计划。
 - `/contine'：如果输出切割，请继续进行。
 - “/语言”：更改AI导师语言

*搜索命令需要插件。
